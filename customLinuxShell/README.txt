INTRODUCTION
------------

Made by: Hriday Kondru

Shell is a UNIX term for the interactive user interface with an operating system. 
The shell is the layer of programming that understands and executes the commands a user enters. 

HOW TO USE
------------

Open the custom_shell.cpp file on a computer running Linux OS.
Compile the .cpp file using the g++ compiler.
There are two ways to run the compiled file in a terminal window:
	without using a text file as input
	specifying the name of the textfile after shell name which runs all the commands one-by-one in the text file.	
Type any supported command/program name with relevant arguements.


SUPPORTED COMMANDS
----------------
clear
env
cd <Directory>
pwd
mkdir <Directory>
rmdir <Directory>
ls
history (or) history<Number of recent entries to display>
exit
<external command> <arguements>
